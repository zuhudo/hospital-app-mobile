# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### 🎉 Initial Release

#### Mobile App (Flutter)
- Splash screen with animations
- Login and registration screens
- Home dashboard with stats and quick actions
- Patient list and detail views
- Appointment booking with calendar
- Doctor directory with filtering
- User profile and settings
- Material Design 3 teal theme
- GoRouter navigation
- Provider state management

#### Backend API (Go + Fiber)
- JWT authentication (login/register)
- Patient CRUD endpoints
- Doctor management endpoints
- Appointment booking and cancellation
- Medical records management
- CORS, logging, and recovery middleware
- In-memory data stores (demo)
- Docker support

#### Public Website (Astro)
- Responsive landing page with hero
- Services showcase
- Doctor profiles
- About page with stats
- Contact form
- Tailwind CSS styling

#### Admin Dashboard (React + Vite)
- Login with authentication
- Dashboard with charts and stats
- Patient management table
- Doctor profiles and cards
- Appointment tracking
- Medical records browser
- Billing overview
- Settings page
- Protected routes
- Zustand state management
- Recharts data visualization
